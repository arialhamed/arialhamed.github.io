UPDATE (2022-Sept-29): ALRIGHT, so i got an mSATA SSD from Carousell, and it seems that the listing's removed, so i think they just had 1 left and i somehow got the last one (that is pretty sick). 128GB, Samsung SM841N, even though its used, its still quite respectable. Ok so that previous paragraph was written around this morning. Just now I found the perfect way to do it. The following is how I did it:
- Get 2.5" SSD & mSATA SSD and install both into X230
- Install Linux on 2.5" SSD and do not touch mSATA in installation
- Shutdown and remove 2.5" SSD and install Windows on the mSATA (btw, very important, if you're gonna install Windows on a ThinkPad, install with Lenovo's disc images, as longer battery times have been reported, probably due to custom drivers (source: Reddit)).
- Ensure Windows boot into mSATA when 2.5" SSD is not inside
- The default boot order for X230 is 2.5" bay then mSATA bay, so you can put the Linux 2.5" drive back, and test the boot. It should boot into Linux

Now by default, my X230 should be able to boot into either OS. I boot into Linux by default, and if i need to boot into Windows, just remove the 2.5" SSD. It's easy because of how the X230 is built, just remove one screw from the bay lid, pull the 2.5" out and turn on the X230 to boot into Windows. This may not be so viable for other laptops, maybe not even other mechanical-gen ThinkPads such as the X250 where it requires removing the entire bottom plate to remove the 2.5" storage (let's not even mention about the hidden 2242 SATA SSD inside the X250)
